## 二、候选选题及淘汰原因

### 选题1：退休对健康的影响（RDD，年龄60岁门槛）

**识别逻辑**：城镇职工强制退休年龄（男60、女55）构成外生断点。

**淘汰原因：第一阶段失效**

CHARLS以农村人口为主。2013波45-75岁男性中：
- 城镇企业养老金领取率 ≈ 0%
- 政府养老金领取率 ≈ 1%
- 新农村养老保险参与率 ≈ 50%（无法区分缴费中与领取中）

实测"不工作"比例在60岁处仅从17.7%跳至22.4%（+4.7pp），远不够RDD要求的锐利断点。
若限定城镇子样本，N过小，精度损失无法接受。

**教训**：RDD可行性必须先验证第一阶段。对于以农村为主的调查，城镇政策门槛的识别力极弱。

---

### 选题2：新农保（NRPS）对健康的影响

**识别逻辑A（DiD）**：Huang & Zhang (2021, AEJ Applied) 利用县级分批推进（2009-2012）的时序差异。

**淘汰原因**：我们的最早波次是2013年，NRPS已覆盖全部县。无处理前基准期，无法构造DiD。

**识别逻辑B（RDD）**：年龄60岁门槛，60岁以上自动领取。

**淘汰原因**：2018年`fn058_w4`（居民/新农保参与）在50-80岁年龄段无明显跳跃（59岁66.9%，60岁69.1%，仅+2.2pp）。该变量混合"缴费中"与"领取中"两类人，无法识别真实的领取跳跃。

**教训**：政策评估需要处理前数据或变量能区分参与状态。

---

### 选题3：COVID-19社区封控对老年心理健康的影响（DiD）

**识别逻辑**：以社区封控强度（`vd001_1`封控天数）为处理，利用2013/2015/2018三期基准检验平行趋势，2020作为处理后期。

**评估**：技术上可行（四期数据、communityID跨波稳定），但：

**放弃原因**：
1. 社区封控强度内生于当地疫情严重程度
2. 无法区分"封控冲击"与"疫情本身冲击"对心理健康的影响
3. 2020波心理健康量表（dc016-dc025）与2013/2015波（dc009-dc018）题目相同但语境不同（2020问的是"春节疫情爆发期间"，早期波问的是"过去一周"），可比性存疑

### 选题4：配偶死亡（丧偶）对幸存者健康的影响（错期DiD）

**识别逻辑**：
- 处理：在某波次间失去配偶（`be001`：在婚→丧偶）
- 不同个体在不同波次间经历丧偶，形成错期处理
- 利用晚期处理组的两个前期（2013、2015）检验平行趋势

**数据核查结果**：

| 指标 | 数值 |
|------|------|
| 三波有效面板N | 14,277 |
| 纳入标准后（基期在婚+三波CES-D完整）| 9,729 |
| 早期处理（2013→2015丧偶）| 193人 |
| 晚期处理（2015→2018丧偶）| 313人 |
| 对照组（全程在婚至2018）| 9,159人 |

**关键变量可用性**：
- 婚姻状态`be001`：2013/2015/2018三波一致，编码稳定（1=在婚，5=丧偶）
- CES-D（10项）：2013/2015在`Health_Status_and_Functioning.dta`的`dc009-dc018`；2018在`Cognition.dta`的`dc009-dc018`，题目完全一致，值域1-4（注意2018波新增8=不知道、9=拒绝，需重编为NA）
- SRH（`da002`）：全四波可用
- ADL/IADL（`db001-db019`）：全四波可用

**淘汰原因：平行趋势正式检验失败（p = 0.028）**

```
正式前趋势检验（晚期处理 vs 对照，2013→2015 CES-D变化量）：

  晚期处理组（N=339）：Δ均值 = +0.912，SD = 6.274
  对照组    （N=9,488）：Δ均值 = +0.149，SD = 5.601
  差值 = +0.762，p = 0.028，95% CI = [0.082, 1.442]
```

在配偶尚未死亡的2013→2015期间，将在2015→2018间丧偶的群体已经表现出**显著更快的抑郁上升趋势**。

**机制解释**：照护负担假说。配偶病情在正式死亡前数年即开始恶化，幸存者承担照护压力，抑郁提前上升，造成"处理前"的趋势差异。这是预期效应（anticipation effect）与选择性（selection）的混合体。

**额外结构性问题**：

| 问题 | 具体表现 |
|------|---------|
| 基期水平不均衡 | 处理组CES-D基期显著高于对照（p < 0.005） |
| 早期处理组仅1个前期 | 193人，完全无法检验平行趋势 |
| 年龄不可比 | 处理组均值63.5岁 vs 对照57.2岁（差6年） |
| 性别不可比 | 处理组仅27-39%男性 vs 对照50% |

**潜在修复方向（已评估，未实施）**：
限定"突发/意外死亡"子样本——2013波`Exit_Interview.dta`中`exb009`记录"死亡是否预期"，限定非预期死亡可部分缓解预期效应问题。但：（1）该变量仅在2013波Exit模块存在，2015/2018波Exit数据未确认；（2）非预期死亡N将进一步收缩；（3）幸存者中"意外"的定义主观性强。

**教训**：丧偶研究的最大内生性不是死亡本身，而是**慢性病配偶的临终照护阶段**会提前污染幸存者健康轨迹。平行趋势在丧偶设计中系统性地面临临终照护渠道的威胁。

## 四、识别策略评估矩阵

| 策略 | 需要什么 | 在CHARLS中是否可行 | 核心障碍 |
|------|---------|-------------------|---------|
| RDD（城镇退休年龄） | 第一阶段跳跃 | ❌ | 农村样本主体，城镇门槛无效 |
| DiD（NRPS县级推进） | 处理前波次 | ❌ | 最早波2013，政策已完成 |
| RDD（NRPS年龄60） | 领取跳跃 | ❌ | 变量不区分缴费/领取状态 |
| 错期DiD（丧偶） | 平行趋势 | ❌ | 正式检验p=0.028，照护预期效应 |
| DiD（COVID封控） | 外生处理 | ⚠️ 有争议 | 封控内生于疫情严重程度，已主动放弃 |
